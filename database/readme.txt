1. Open CLI
2. Start mysql server
3. Create a new database with any name of your choice
	CREATE database name;
4. Open new CLI window
5. Run command
	mysql -u root -p database_name (that you created in step 3) < file_path/asur.sql
6. Open mysql
7. SHOW databases;
8. USE database_name; (that you created in step 3)
9. You will find asur.sql imported into your database.

- asur.ipynb to load and retrieve image file into student table, can also be used to retrieve data
